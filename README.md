# deltarune en galego

<details>
<summary>galego</summary>

parche completo dos dous primeiros capítulos (gratis!) de *deltarune* ao idioma galego.

### instalación

- procura a última versión no apartado de "Releases"
- descarga o arquivo deltagal.zip
- descomprime o contido no directorio **DELTARUNE** de steam  
  (pódese acceder desde steam usando “abrir arquivos locais” nas opcións do xogo)
- substituír todos os arquivos cando se solicite. aínda que xa teñas unha version vella do patch instalado ou unha partida comezada, non pasa nada!! isto non borra en absoluto o teu progreso no xogo. as partidas de deltarune gardanse nun directorio distinto. podes ter unha partida comezada cunha version do patch, actualizalo, e seguir xogando na mesma partida.
- ao iniciar o xogo por primeira vez, preguntarase "Would you like to start in chapter 1?", seleccionas Yes

- na seguinte pantalla, unha das opcións de abaixo será CONFIG
- aí, seleccionas o ENGLISH e, coas frechas de dereita/esquerda, cambias o idioma a galego!

### notas legais

as aclaracións legais están incluídas no ficheiro de licenza. todo o contido orixinal creado por min é de uso completamente libre sen necesidade de dar crédito.

### merda

este é un proxecto ao que lle dediquei moitísimo tempo, e estou satisfeito co resultado.  
gustaríame que servise para achegar un dos meus xogos favoritos a máis xente, ou para axudar a cubrir o baleiro existente no mundo dos videoxogos en canto a traducións oficiais ao galego.

se o vas probar, moitas grazas, e espero que o desfrutes :)

## english

complete patch for the first two chapters (free!) of *deltarune*, translated into galician.

### installation

- get the latest release from github
- download deltagal.zip
- extract into the **DELTARUNE** steam directory  
  (this can be accessed from steam by selecting “browse local files” in the game options)
- replace all files when prompted
- when launching chapter 1, a **config** option should appear, allowing you to switch to galician

### legal notes

legal clarifications are included in the license file. all included files are free to use with no credit required.

### bullshit

my hope is that it helps bring one of my favorite games to more people, or helps fill the gap in the videogame space when it comes to official galician localizations.

if you decide to try it, thank you very much, and i hope you enjoy it :)
